'RideWeather' Weather Application:

A single page application developed using Preact and the OpenWeather API.

Collaborators:

- Darren Ohonba
- Yuhan	Jiang
- Dilkush Punja

Developed in:

- Node v16.14.0
- npm  v8.3.1
- Preact v6.4.0
- HTML5
- CSS3

Instructions for running the application:

1. Install node and npm.
2. Using your terminal environment navigate to the directory containing the 'src' folder and enter 'npm install' followed by 'npm run dev'.
3. Use a web browser to navigate to the local address once the development server is running.

Enjoy!
